# Таргет: RPICO_RP2040

Компактная плата на RP2040 (USB-C, два ядра Cortex-M0+ @ 125 МГц, 264 КБ SRAM, 2 МБ Flash).
Совместима со стандартной распиновкой Raspberry Pi Pico.
ID платы в PlatformIO: **`rpipico`**.

Справочная схема распиновки (сохранить изображение рядом с этим файлом как `board_pinout.png`):

```
                         ┌─────USB-C─────┐
                    LED ◄─ GP25          │
                         │               │
  CRSF TX  UART0_TX  GP0 ┤             ├ VREF
  CRSF RX  UART0_RX  GP1 ┤  RP2040     ├ Vout
                     GND ┤             ├ Vin / GND
  L_RPWM   PWM1A     GP2 ┤             ├ GP23
  L_LPWM   PWM1B     GP3 ┤             ├ 3V3
  L_REN    GPIO OUT  GP4 ┤             ├ GP29  ADC3
  L_LEN    GPIO OUT  GP5 ┤  ┌──────┐   ├ GP28  ADC2  VBAT
                     GND ┤  │      │   ├ AGND
  R_RPWM   PWM3A     GP6 ┤  │      │   ├ GP27  ADC1  IS_LEFT
  R_LPWM   PWM3B     GP7 ┤  └──────┘   ├ GP26  ADC0  IS_RIGHT
  R_REN    GPIO OUT  GP8 ┤             ├ GP22  IMU INT1
  R_LEN    GPIO OUT  GP9 ┤             ├ GP21  I2C0_SCL  IMU
                     GND ┤             ├ GP20  I2C0_SDA  IMU
  —              —     GP10 ┤             ├ GP19
  —              —     GP11 ┤     RGB     ├ GP18
  SBC TX   UART1_TX  GP12 ┤             ├ GP17
  SBC RX   UART1_RX  GP13 ┤             ├ GP16
                      GND ┤             ├ GND
  —              —     GP14 ┤             ├ GP17
  —              —     GP15 ┤             ├ GP16  —
                    3V3 ─┘             └─ GND
                        SWDIO       SWCLK
```

## Распиновка по группам периферии

### 1. CRSF / ELRS приёмник (UART0)

| Пин | Сигнал   | Направление | Примечание                   |
|-----|----------|-------------|------------------------------|
| GP0 | CRSF_TX  | ВЫХ UART0  | К пину RX приёмника          |
| GP1 | CRSF_RX  | ВХ  UART0  | От пина TX приёмника         |

### 2. Драйвер левого мотора BTS7960

| Пин | Сигнал | Направление | Примечание                   |
|-----|--------|-------------|------------------------------|
| GP2 | L_RPWM | ВЫХ ШИМ    | ШИМ слайс 1, канал A         |
| GP3 | L_LPWM | ВЫХ ШИМ    | ШИМ слайс 1, канал B         |
| GP4 | L_REN  | ВЫХ GPIO   | Разрешение правой полумост   |
| GP5 | L_LEN  | ВЫХ GPIO   | Разрешение левой полумост    |

### 3. Драйвер правого мотора BTS7960

| Пин | Сигнал | Направление | Примечание                   |
|-----|--------|-------------|------------------------------|
| GP6 | R_RPWM | ВЫХ ШИМ    | ШИМ слайс 3, канал A         |
| GP7 | R_LPWM | ВЫХ ШИМ    | ШИМ слайс 3, канал B         |
| GP8 | R_REN  | ВЫХ GPIO   | Разрешение правой полумост   |
| GP9 | R_LEN  | ВЫХ GPIO   | Разрешение левой полумост    |

### 4. Связь с одноплатным компьютером (UART1)

| Пин  | Сигнал   | Направление | Примечание                         |
|------|----------|-------------|------------------------------------|
| GP12 | UART1_TX | ВЫХ         | К пину RX одноплатного компьютера  |
| GP13 | UART1_RX | ВХ          | От пина TX одноплатного компьютера |

### 5. IMU (I2C0)

| Пин  | Сигнал   | Направление | Примечание                         |
|------|----------|-------------|------------------------------------|
| GP20 | I2C0_SDA | I/O         | Шина данных I2C0                   |
| GP21 | I2C0_SCL | ВЫХ         | Тактовый I2C0                      |
| GP22 | IMU_INT1 | ВХ  GPIO   | Прерывание от IMU                  |

### 6. АЦП — датчики тока и напряжение батареи

| Пин  | Канал | Сигнал   | Примечание                                              |
|------|-------|----------|---------------------------------------------------------|
| GP26 | ADC0  | IS_RIGHT | RC-фильтр 1kΩ‖1kΩ + 0.1µF; IS-пины BTS7960 правого     |
| GP27 | ADC1  | IS_LEFT  | RC-фильтр 1kΩ‖1kΩ + 0.1µF; IS-пины BTS7960 левого      |
| GP28 | ADC2  | VBAT     | Делитель напряжения; коэффициент — `BIBA_VBAT_DIVIDER_RATIO` |

### 7. Индикация (встроенная)

| Пин  | Сигнал     | Примечание                                        |
|------|------------|---------------------------------------------------|
| GP23 | RGB_LED    | WS2812 NeoPixel, встроенный на YD-RP2040          |
| GP25 | STATUS_LED | Встроенный светодиод Pico, активный HIGH           |

### 8. Свободные пины (выведены на плате для расширения)

Следующие пины ничем не заняты и разведены на отдельный разъём / площадки на печатной плате.
Могут быть назначены на любую периферию в любой момент.

| Пин  | Аппаратные функции RP2040         | Возможное применение              |
|------|-----------------------------------|-----------------------------------|
| GP10 | SPI1_SCK / UART1_TX / PWM5A / I2C1_SDA | GPIO, SPI, доп. UART, ШИМ   |
| GP11 | SPI1_TX  / UART1_RX / PWM5B / I2C1_SCL | GPIO, SPI, доп. UART, ШИМ   |
| GP14 | SPI1_SCK / UART0_TX / PWM7A / I2C1_SDA | GPIO, ШИМ, I2C              |
| GP15 | SPI1_TX  / UART0_RX / PWM7B / I2C1_SCL | GPIO, ШИМ, I2C              |
| GP16 | SPI0_RX  / UART0_TX / PWM0A / I2C0_SDA | GPIO, ШИМ                   |
| GP17 | SPI0_CSn / UART0_RX / PWM0B / I2C0_SCL | GPIO, ШИМ                   |
| GP18 | SPI0_SCK / UART0_TX / PWM1A / I2C1_SDA | GPIO, ШИМ                   |
| GP19 | SPI0_TX  / UART0_RX / PWM1B / I2C1_SCL | GPIO, ШИМ                   |
| GP29 | ADC3                              | АЦП (0–3.3 В), GPIO              |

## Частота несущей ШИМ


```
pio run -e rpico_rp2040_standalone
pio run -e rpico_rp2040_companion
```

Прошивка через USB (зажать BOOTSEL, подключить USB, отпустить BOOTSEL):
```
pio run -e rpico_rp2040_standalone --target upload
```
