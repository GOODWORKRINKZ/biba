# biba_remote_bridge (hook-point)

> Управление через интернет: Zenoh-bridge / MQTT / WebRTC / VPN. Пустая директория с `COLCON_IGNORE`.

Целевая роль: исходящий мост ROS2-стека композиции C во внешнюю сеть. Канонический паттерн — Zenoh-router с подключением к облачному peer'у (см. `rob_box_project`).
