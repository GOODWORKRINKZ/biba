  Script header:
    #!/usr/bin/env python3
    """Load gate threshold grid search — validates LOAD_RATIO_THRESH and LOAD_QUALITY_MAX
    against softhold capture before firmware implementation."""

  Imports: pathlib.Path, sys, numpy as np, pandas as pd, matplotlib (use("Agg")), matplotlib.pyplot as plt, scipy.stats (pearsonr not needed here — import only what is used).

  Constants block (at module level, after imports):
    BASE_DIR = Path(__file__).resolve().parent / "artifacts" / "is-sweepraw"
    CSV_STEM = "sweepraw_TRAP_amp50_per6000_n60_20260526-135642_softhold"
    OUT_PNG  = Path(__file__).resolve().parent / "artifacts" / "load_gate_threshold_grid.png"
    THRESH_GRID   = [1.2, 1.5, 1.8, 2.0, 2.5]
    QMAX_GRID     = [5.0, 7.0, 10.0, 12.0]
    ABS_THRESH_DEFAULT = 3800.0   # fallback absolute gate (from D-A3)

  Data loading:
    Read BASE_DIR / (CSV_STEM + "_left.csv") and BASE_DIR / (CSV_STEM + "_right.csv")
    into df_l, df_r with pandas.read_csv.
    Compute per-window mean_adc via groupby("win_idx")["adc_raw"].mean() → Series dc_l, dc_r indexed by win_idx.
    Merge on win_idx into merged DataFrame with columns: win_idx, dc_l, dc_r.

  Load quality lookup:
    The softhold CSV does not contain quality scores; they come from the research notes.
    Hard-code the three sentinel windows as a dict:
      SENTINEL_WINS = {
          3:  {"dc_l": 2588.0, "dc_r": 1383.0, "quality": 3.7,  "expected": "REJECT"},
          14: {"dc_l": 1139.0, "dc_r":  860.0, "quality": 11.1, "expected": "KEEP"},
          18: {"dc_l": 3586.0, "dc_r": 1503.0, "quality": 9.4,  "expected": "REJECT"},
      }
    The script uses the sentinel dict for classification checks (not the CSV means, which are
    computed from all 1024 samples per window and may differ slightly from the doc values; the
    sentinel values are the ground truth for gate validation).

  Gate function (exact signature per D-A2 + D-A3):
    def apply_load_gate(dc_primary: float, dc_other: float,
                        quality: float,
                        ratio_thresh: float, quality_max: float,
                        abs_thresh: float = ABS_THRESH_DEFAULT) -> bool:
        """Return True if window should be REJECTED (HIGH_LOAD)."""
        ratio = dc_primary / (dc_other + 1e-6)
        if ratio > ratio_thresh and quality < quality_max:
            return True
        if dc_primary > abs_thresh:
            return True
        return False

  Grid search:
    Iterate over all (thresh, qmax) combinations (5×4=20 total).
    For each combo, classify all three sentinel windows using apply_load_gate(dc_l, dc_r, quality, thresh, qmax).
    Record result as "REJECT" (gate returns True) or "KEEP" (gate returns False).
    Mark combo as "correct" if win3→REJECT AND win18→REJECT AND win14→KEEP.
    Collect correct combos. From correct combos select optimal = minimum THRESH among correct;
    if tie, choose maximum QMAX (per planning requirement: min THRESH + max QUALITY_MAX).

  Print output (exact format):
    For each combo print one line:
      "  THRESH={t:.1f}  QMAX={q:.1f}  win3={r3}  win18={r18}  win14={r14}  {'OK' if correct else 'FAIL'}"
    After grid, print blank line, then:
      "CONFIRMED: LOAD_RATIO_THRESH={opt_t:.1f} LOAD_QUALITY_MAX={opt_q:.1f}"
    If no combo is correct, print:
      "ERROR: no valid threshold found — review sentinel data"
    and sys.exit(1).

  Figure:
    Create a figure with 2 subplots side-by-side (figsize=(12, 5)).
    Left subplot: 2D grid heatmap (rows=QMAX_GRID, cols=THRESH_GRID).
      Cell value = 1 if correct, 0 if not. Use imshow with cmap="RdYlGn", vmin=0, vmax=1.
      Mark optimal cell with a white star (*) annotation.
      Title: "Load Gate Grid (green=correct)"
      x-label: "LOAD_RATIO_THRESH", y-label: "LOAD_QUALITY_MAX"
      Set xticks/yticks to THRESH_GRID / QMAX_GRID.
    Right subplot: scatter of (ratio, quality) for all 60 windows from merged CSV, coloured by
      whether optimal gate rejects them. Overlay the 3 sentinel windows with larger markers and labels
      "win3", "win14", "win18". Draw horizontal line at optimal QMAX, vertical line at optimal THRESH.
      Title: "Sentinel Windows — optimal gate boundary"
      x-label: "ratio (dc_l / dc_r)", y-label: "quality (from bench)"
      Note: quality for all 60 windows is NOT in the CSV — omit quality scatter; plot only DC ratio
      scatter (y=dc_l) coloured by gate decision using optimal thresholds. Use x=ratio, y=dc_l.
    Save to OUT_PNG with bbox_inches="tight", dpi=150.
    Print: "saved: {OUT_PNG}"
