    #!/usr/bin/env python3
    """Throttle vs load disambiguation — inter-window (Δfreq, ΔDC) gradient analysis."""

  Imports: pathlib.Path, sys, numpy as np, pandas as pd, matplotlib (use("Agg")), matplotlib.pyplot as plt.
  Optional (use try/except): from sklearn.discriminant_analysis import LinearDiscriminantAnalysis.

  Constants:
    BASE_DIR  = Path(__file__).resolve().parent / "artifacts" / "is-sweepraw"
    CSV_STEM  = "sweepraw_TRAP_amp50_per6000_n60_20260526-135642_softhold"
    OUT_PNG   = Path(__file__).resolve().parent / "artifacts" / "load_disambiguate_scatter.png"
    ADR_PATH  = Path(__file__).resolve().parent.parent / ".planning" / "phases" / \
                "11-is-pin-load-stall-detection" / "11-LOAD-DISAMBIGUATE-ADR.md"
    SPS       = 10000
    N_SAMPLES = 1024

  Data loading:
    Read CSV_STEM + "_left.csv" into df_l.
    Compute per-window dc_l (mean adc_raw per win_idx) as before.

  Frequency estimation per window:
    The softhold CSV does not contain spectral results. Compute freq_hz for each window using
    a simplified DFT peak search (same logic as is_algo_bench.py / is_sweepraw_analyze.py):
    For each win_idx in sorted order:
      samples = df_l[df_l["win_idx"]==w]["adc_raw"].values  (numpy array)
      Compute FFT: X = np.fft.rfft(samples - samples.mean())
      freqs = np.fft.rfftfreq(len(samples), d=1.0/SPS)
      Search band [100, 600] Hz for peak: mask = (freqs >= 100) & (freqs <= 600)
      peak_bin = np.argmax(np.abs(X[mask]))
      freq_hz = freqs[mask][peak_bin]
      Record (win_idx, freq_hz, dc_l).

    Collect into DataFrame: df_wins with columns [win_idx, freq_hz, dc_l].

  Inter-window deltas:
    Sort df_wins by win_idx.
    delta_freq = df_wins["freq_hz"].diff()    # NaN for first row
    delta_dc   = df_wins["dc_l"].diff()
    Drop first row (NaN). Add delta_freq, delta_dc as columns.

  Category assignment (per D-D2 hypothesis):
    def classify(df_row) -> str:
        dfreq, ddc = df_row["delta_freq"], df_row["delta_dc"]
        if abs(dfreq) < 20 and ddc > 500:
            return "stall"
        elif dfreq > 10 and ddc > 50:
            return "acceleration"
        elif dfreq < -10 and ddc > 50:
            return "load"
        else:
            return "steady"

    Apply to df_wins (after dropping NaN row). Column: "category".

  Separability:
    Count windows per category and print:
      f"Category counts: {df_wins['category'].value_counts().to_dict()}"
    Try LDA accuracy (if sklearn available): fit on [delta_freq, delta_dc], predict, report accuracy.
    Print:
      "LDA separability accuracy: {acc:.2f}" if sklearn available
      "Note: install scikit-learn for LDA separability score" otherwise.

  Print summary lines:
    f"Windows analysed: {len(df_wins)}"
    For each category, print the win_idx list.
    "Decision boundary hypothesis (D-D2):"
    "  acceleration: d_freq > 0 AND d_DC > 0"
    "  load:         d_freq < 0 AND d_DC > 0"
    "  stall:        |d_freq| → 0 AND d_DC >> 0"

  Figure:
    figsize=(8, 6)
    Color map: {"acceleration": "green", "load": "orange", "stall": "red", "steady": "grey"}
    For each category, scatter delta_freq (x) vs delta_dc (y) with label and color.
    Add vertical dashed line at x=0 (sign boundary for freq), horizontal dashed line at y=0.
    x-label: "Δfreq (Hz/window)", y-label: "ΔDC (ADC counts/window)"
    Title: "Throttle vs Load Gradient Disambiguation — softhold dataset"
    legend(loc="upper right")
    Save to OUT_PNG, bbox_inches="tight", dpi=150.
    Print: f"saved: {OUT_PNG}"

  # ADR: Throttle vs Load Disambiguation (Phase 11)

  **Date:** {datetime.date.today().isoformat()}
  **Status:** Research
  **Phase:** 11 — IS-Pin Load & Stall Detection

  ## Context

  The spectral RPM estimator provides per-window freq_hz and quality. The IS-pin DC mean
  (mean_adc) is available after Phase 11's load gate extension. Together these two signals
  form a 2D feature vector (Δfreq, ΔDC) per inter-window transition that may distinguish
  three operating modes:
  - **Acceleration:** motor spinning up (d_freq > 0 AND d_DC > 0 — more current, rising RPM)
  - **Load increase:** external torque applied (d_freq < 0 AND d_DC > 0 — more current, falling RPM)
  - **Stall:** motor stopped (|d_freq| ≈ 0 AND d_DC >> 0 — maximum current, no rotation)

  ## Research Results

  Dataset: sweepraw_TRAP_amp50_per6000_n60_20260526-135642_softhold (60 windows, TRAP 50%)

  Category counts: {category_counts}

  LDA separability: {lda_result}

  ## Decision

  **D-D2 hypothesis confirmed / partially confirmed / not confirmed** (fill in after running script).

  ## Proposed Detection Rule (Phase 12+ firmware target)

  ```
  if |d_freq| < 20 Hz AND d_DC > 500 counts:
      state = STALL
  elif d_freq > 10 Hz AND d_DC > 50 counts:
      state = ACCELERATION
  elif d_freq < -10 Hz AND d_DC > 50 counts:
      state = LOAD_INCREASE
  else:
      state = STEADY
  ```
