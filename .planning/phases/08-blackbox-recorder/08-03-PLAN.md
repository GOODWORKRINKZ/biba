Function auto_detect_port() -> str:
  Use glob.glob("/dev/ttyACM*") to find candidate ports; if exactly one found, return it;
  if multiple found, return the first (lowest numbered); if none, raise RuntimeError with
  message "No /dev/ttyACM* device found. Is the RP2040 connected?".

Function list_sessions(port: serial.Serial) -> list[str]:
  Write b"bb list\n" to port. Read lines until a blank line or timeout (set port.timeout=2).
  Parse each line as "session_NNNN.bbd N bytes" using regex r"(session_\d{4}\.bbd)".
  Return list of filenames. If no sessions, return empty list.

Function download_session(port: serial.Serial, filename: str, out_dir: Path) -> bytes:
  Validate filename matches regex r"^session_\d{4}\.bbd$" — raise ValueError if not
  (client-side path traversal guard mirroring the firmware-side check).
  Write f"bb get {filename}\n".encode() to port.
  Read the SIZE header line: port.readline() → decode → strip → match r"SIZE:(\d+)".
  If line starts with "ERR:", raise RuntimeError(line).
  size = int(match.group(1)).
  Read exactly size bytes: loop port.read(size - len(data)) until len(data) == size.
  Write raw bytes to out_dir / filename.
  Return data bytes.

Function decode_bbd(data: bytes) -> tuple[dict, list[dict]]:
  Unpack 32-byte header: struct.unpack_from("<4sIHB", data, 0) → magic, created_ms,
  field_mask, rate_hz. Assert magic == b"BBD1", raise ValueError if not.
  Build active fields list from field_mask bits.
  Compute rec_fmt = "<" + joined fmt chars of active fields.
  rec_size = struct.calcsize(rec_fmt).
  Parse records from offset 32 until data exhausted.
  Return (header_dict, records_list) where header_dict has keys: magic (str), created_ms,
  field_mask (hex string), rate_hz.

Function save_csv(filename: str, records: list[dict], out_dir: Path) -> Path:
  csv_path = out_dir / filename.replace(".bbd", ".csv").
  Write with csv.DictWriter using fieldnames from first record keys (if records non-empty).
  If records empty, write header-only CSV with all 16 field names.
  Return csv_path.

Function main() -> int:
  argparse setup:
    --port: serial port (default: auto-detect)
    --baud: int default 115200
    --all: flag, download all sessions
    --session NNNN: specific 4-digit session number (e.g. "0001")
    --no-decode: flag, skip CSV decode (save .bbd only)
  OUT_DIR.mkdir(parents=True, exist_ok=True).
  Determine port: args.port or auto_detect_port().
  Open serial.Serial(port, baud, timeout=5).
  Send "\n" once and drain to clear any partial line in firmware buffer; sleep 0.1s.
  If --all: sessions = list_sessions(); download each.
  If --session NNNN: filename = f"session_{args.session:0>4}.bbd"; download it.
  If neither flag: print list of sessions without downloading (same as bb list).
  For each downloaded session (unless --no-decode): decode_bbd() then save_csv().
  Print summary: "Downloaded N session(s) to {OUT_DIR}".
  Return 0.

if __name__ == "__main__": sys.exit(main()).

Security: both auto_detect_port and download_session are the only places user/device input
reaches filesystem or serial — both validated. The port path comes from glob (controlled
device name pattern), not from user string. The filename is validated with regex before use.
