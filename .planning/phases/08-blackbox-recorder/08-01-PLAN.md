  <name>Task 1: blackbox.h — struct definitions and C API declarations</name>
  <files>firmware/src/app/blackbox.h</files>
  <action>
Create firmware/src/app/blackbox.h as a C-compatible header (no C++ headers, guarded with
#ifdef __cplusplus extern "C" { #endif).

Define BLACKBOX_MAGIC as the 4-byte literal "BBD1".

Define biba_blackbox_header_t as a __attribute__((packed)) struct with exactly these fields
in order: uint8_t magic[4], uint32_t created_tick_ms, uint16_t field_mask, uint8_t rate_hz,
uint8_t reserved[21]. Total sizeof must be 32 bytes. Add a static_assert to enforce this.

Define biba_blackbox_record_t as a __attribute__((packed)) struct with all 16 log fields in
bit-index order: uint32_t timestamp_ms (bit 0); int16_t throttle (bit 1, scaled ×1000);
int16_t rudder (bit 2, ×1000); int16_t duty_left (bit 3, ×1000); int16_t duty_right
(bit 4, ×1000); uint16_t rpm_left_hz10 (bit 5); uint16_t rpm_right_hz10 (bit 6);
uint8_t active_blocks_l (bit 7); uint8_t active_blocks_r (bit 8); uint16_t mean_is_l
(bit 9); uint16_t mean_is_r (bit 10); uint8_t latch_resets (bit 11); uint16_t vbat_mv
(bit 12); int16_t pi_integral_l (bit 13, ×10000); int16_t pi_integral_r (bit 14, ×10000);
uint16_t pi_meas_ema_l (bit 15, ×10).

Total sizeof(biba_blackbox_record_t) must be 31 bytes. Add a static_assert for this too.

Declare these extern "C" functions (no implementations here):
  bool blackbox_init(void);
  bool blackbox_open_session(uint32_t session_num, uint32_t tick_ms,
                             uint16_t field_mask, uint8_t rate_hz);
  void blackbox_write_record(const uint8_t *buf, size_t len);
  void blackbox_close_session(void);
  bool blackbox_is_full(uint32_t min_free_kb);
  uint32_t blackbox_next_session_num(uint32_t *oldest_out);
  void blackbox_delete_oldest(void);
  void blackbox_list_sessions(void);
  bool blackbox_send_session(const char *filename);
  bool blackbox_delete_session(const char *filename);
  void blackbox_info(void);

The last four functions are needed by the CDC shell in Plan 03 — declare them here so
blackbox.cpp can export them all in one file.

Include guards: BIBA_BLACKBOX_H.
Standard includes: stdint.h, stddef.h, stdbool.h.

<task type="auto">
  <name>Task 2: blackbox.cpp — LittleFS wrapper + biba_hal_serial write_bytes</name>
  <files>firmware/src/app/blackbox.cpp, firmware/src/hal/biba_hal.h, firmware/src/hal/biba_hal_serial.cpp</files>
  <action>
Create firmware/src/app/blackbox.cpp. Follow the structure of biba_hal_serial.cpp exactly
(include "biba_hal.h"-style, then Arduino header, then extern "C" implementations).

At top: #include "blackbox.h" then #include &lt;LittleFS.h&gt; then #include &lt;Arduino.h&gt;
then #include &lt;stdio.h&gt; and #include &lt;string.h&gt;.

File-scope statics: static File s_session_file; static bool s_mounted = false;

Implement blackbox_init: call LittleFS.begin(); if it fails, call LittleFS.format() then
LittleFS.begin() again; set s_mounted to the result; return s_mounted.

Implement blackbox_open_session: build path as "/session_%04lu.bbd" from session_num;
open with LittleFS.open(path, "w"); if open fails return false; build a
biba_blackbox_header_t with magic "BBD1", created_tick_ms, field_mask, rate_hz, all
reserved bytes zeroed; write sizeof(hdr) bytes to file; return true.

Implement blackbox_write_record: if s_session_file is truthy, call
s_session_file.write(buf, len).

Implement blackbox_close_session: if s_session_file is truthy, call
s_session_file.flush() then s_session_file.close().

Implement blackbox_is_full: call LittleFS.info(fs_info); return
(fs_info.totalBytes - fs_info.usedBytes) < (min_free_kb * 1024UL).
Declare FSInfo fs_info as a local variable.

Implement blackbox_next_session_num: open Dir with LittleFS.openDir("/"); iterate
dir.next(); for each entry, sscanf dir.fileName().c_str() with format "session_%04u.bbd"
into unsigned n; track min_n and max_n across all matches; write min_n to *oldest_out if
not NULL; return max_n + 1 if any found, else 1.

Implement blackbox_delete_oldest: call blackbox_next_session_num(&oldest) to get the
oldest number; build path "/session_%04lu.bbd"; call LittleFS.remove(path).

Implement blackbox_list_sessions: iterate LittleFS.openDir("/") the same way; for each
session_NNNN.bbd entry, call dir.fileSize() and printf "%s %lu bytes\r\n".

Implement blackbox_send_session: validate that filename matches the pattern
"session_%04u.bbd" strictly using sscanf (reject any filename that doesn't parse cleanly
to exactly that pattern — path traversal mitigation, per security note T-08-01 in threat
model). Build full path as "/" + filename. Open with LittleFS.open(path, "r"); if not
found, print "ERR:NOT_FOUND\r\n" and return false. Get file size via file.size(); print
"SIZE:%lu\r\n" (ASCII). Then read and send all bytes through biba_hal_serial_write_bytes()
in 256-byte chunks. Close file. Return true. Never call this while the robot is armed —
enforcement is the caller's responsibility; document with a comment.

Implement blackbox_delete_session: validate filename with same sscanf guard as
blackbox_send_session. Build path "/". + filename. Call LittleFS.remove(path).
Print "OK\r\n" on success, "ERR:NOT_FOUND\r\n" on failure.

Implement blackbox_info: call LittleFS.info(fs_info); printf "FS total=%lu used=%lu
free=%lu bytes\r\n". Also scan directory to count session files; printf "Sessions: %u\r\n".

Add to firmware/src/hal/biba_hal.h (after the existing biba_hal_serial_readline
declaration): void biba_hal_serial_write_bytes(const uint8_t *buf, size_t len);
and void biba_hal_serial_write_str(const char *s);

Add to firmware/src/hal/biba_hal_serial.cpp (append after the existing
biba_hal_serial_readline function):
  extern "C" void biba_hal_serial_write_bytes(const uint8_t *buf, size_t len) {
    Serial.write(buf, len);
  }
  extern "C" void biba_hal_serial_write_str(const char *s) {
    Serial.print(s);
  }
