CH8 enable/disable detection: compute bb_ch_active as bool from

with a static bool s_bb_ch_prev (local to the block or file-scope static).

On CH8 rising edge (bb_ch_active && !s_bb_ch_prev):
  1. If blackbox_is_full(BIBA_BLACKBOX_MIN_FREE_KB):
       if s_bb_full_warned: call blackbox_delete_oldest();
                            s_bb_full_warned = false;
                            printf("[biba] BB: deleted oldest session\r\n");
       else: biba_melody_player_start(&s_player, &biba_melody_failsafe);
             s_bb_full_warned = true;
             printf("[biba] BB: flash full, CH8 again to delete oldest\r\n");
             do NOT set s_bb_enabled (block enable until operator confirms).
             goto/skip the enable logic below.
  2. If NOT full (or after delete above): s_bb_enabled = true; s_bb_full_warned = false;
     biba_melody_player_start(&s_player, &biba_melody_sos);
     printf("[biba] BB: enabled\r\n").

On CH8 falling edge (!bb_ch_active && s_bb_ch_prev):
  s_bb_enabled = false;
  if (s_bb_recording): blackbox_close_session(); s_bb_recording = false;
                       printf("[biba] BB: session closed (CH8 LOW)\r\n").
  printf("[biba] BB: disabled\r\n").

Update s_bb_ch_prev = bb_ch_active at end of the block.

Arm edge detection: in the existing arm-edge block (where s_armed transitions from false
to true), add:
  if (s_bb_enabled && !blackbox_is_full(BIBA_BLACKBOX_MIN_FREE_KB)) {
      s_bb_session_num = blackbox_next_session_num(NULL);
      if (blackbox_open_session(s_bb_session_num, now,
                                BIBA_BLACKBOX_FIELD_MASK, BIBA_BLACKBOX_RATE_HZ)) {
          s_bb_recording = true;
          s_bb_next_ms   = now;
          printf("[biba] BB: session %04lu opened\r\n", (unsigned long)s_bb_session_num);
      }
  }

Disarm edge detection: in the existing disarm-edge block (s_armed transitions from true
to false), add:
  if (s_bb_recording) {
      blackbox_close_session();
      s_bb_recording = false;
      printf("[biba] BB: session %04lu closed\r\n", (unsigned long)s_bb_session_num);
  }

update, add the rate-throttled write block:
  if (s_bb_recording && (int32_t)(now - s_bb_next_ms) >= 0) {

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| DMA/ADC IRQ → main tick | mean_is values cross from ISR to main context |
| main tick → LittleFS (via blackbox.cpp) | filesystem writes happen from main context only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Integrity | s_mean_is_left/right volatile promotion | mitigate | Declared volatile; main-context read is naturally atomic for uint16_t on ARM Cortex-M0+. No additional barrier needed. |
| T-08-05 | Denial of Service | blackbox_write_record() blocking on flash erase (45 ms stall every ~13 s) | accept | Documented in research: 45 ms stall is within CRSF failsafe timeout (500 ms). Control loop stalls briefly but recovers. Acceptable in flight context. |
| T-08-06 | Integrity | Record write while disarmed | mitigate | s_bb_recording is only set true on arm edge, cleared on disarm edge or CH8 LOW. The per-tick write block is gated on s_bb_recording. |
</threat_model>
