    1. `void biba_ramp_init(biba_ramp_t *r)`:
       Guard `if (r == NULL) return;`. Set `r->current = 0.0f;` and `r->hold_remaining_s = 0.0f;`.

    2. `void biba_ramp_reset(biba_ramp_t *r)`:
       Guard `if (r == NULL) return;`. Set `r->current = 0.0f;` and `r->hold_remaining_s = 0.0f;`.
       Comment: `/* D-04: hard reset — emergency stop, no gradual decel */`

    3. `float biba_ramp_update(biba_ramp_t *r, float target, float dt)`:
       CRITICAL guard: `if (dt <= 0.0f) return r->current;` as FIRST statement (Pitfall 1).
       Clamp target: `if (target > 1.0f) target = 1.0f; if (target < -1.0f) target = -1.0f;`
       Zero-hold block: if `r->hold_remaining_s > 0.0f`, subtract dt; if still > 0 return r->current;
         else set hold_remaining_s = 0.0f.
       Direction-change block: if (r->current > 0 && target < 0) OR (r->current < 0 && target > 0):
         max_step = BIBA_RAMP_REVERSE_DECEL_RATE * dt
         abs_cur = (r->current < 0.0f) ? -r->current : r->current
         if abs_cur <= max_step: r->current = 0.0f; r->hold_remaining_s = (float)BIBA_RAMP_ZERO_HOLD_MS / 1000.0f;
         else if r->current > 0: r->current -= max_step; else r->current += max_step;
         return r->current;
       Same-sign acceleration/deceleration:
         diff = target - r->current; abs_diff = (diff<0)?-diff:diff;
         if abs_diff < 1e-9f return r->current;
         abs_target = (target<0)?-target:target; abs_current = (r->current<0)?-r->current:r->current;
         accelerating = (abs_target > abs_current);
         rate = accelerating ? BIBA_RAMP_ACCEL_RATE : BIBA_RAMP_DECEL_RATE;
         max_step = rate * dt;
         if abs_diff <= max_step: r->current = target;
         else r->current += (diff > 0.0f) ? max_step : -max_step;
       Final clamp: `if (r->current > 1.0f) r->current = 1.0f; if (r->current < -1.0f) r->current = -1.0f;`
       return r->current;

    Includes required:
      `#include "ramp.h"`
      `#include "biba_config.h"`
