  def spectral_estimate_hint(buf: np.ndarray, sps: int,
                              target_hz: float, hint_hz: float = 0.0) -> SpectralResult:

  Logic (per D3, D5, D4 from SPEC):
  1. Compute r_plant = spectral_estimate(buf, sps, target_hz)  ← plant window always runs
  2. Guard: if hint_hz == 0.0 or abs(hint_hz - target_hz) <= _DEADBAND_HZ → return r_plant
     (sentinel 0.0 = suppressed per D3; within deadband = not worth second pass per D5)
  3. Compute r_hint = spectral_estimate(buf, sps, hint_hz)   ← hint_hz used as centre
  4. Best-of-two (per D4): if r_hint.valid and r_hint.peak_amp > r_plant.peak_amp:
       r_hint.reason = "hint"   ← maps to HINT_MEASURED in firmware
       return r_hint
  5. Otherwise return r_plant

  Important: spectral_estimate() already clips target_hz to [_SPEC_MIN_HZ, _SPEC_MAX_HZ]
  internally, so passing hint_hz directly is safe (mirrors firmware clampf_local behaviour).
  </action>

  def run_simulation(csv_path: Path, hint_reset_on_dir: bool) -> dict:
    Load windows via load_sweepraw(csv_path).
    Iterate windows in order. Per window:
      - Skip if duty <= 0 (FWD only per D6; duty field from w["duty"])
      - Compute target_hz = expected_hz(w["duty"])
      - Determine direction sign change: compare sign(duty) with prev_duty sign
      - If hint_reset_on_dir=True and direction changed: hint_hz = 0.0
      - buf = w["samples"].astype(np.float32)
      - r_orig = spectral_estimate(buf, SPS, target_hz)            ← single-window baseline
      - r_hint = spectral_estimate_hint(buf, SPS, target_hz, hint_hz)  ← dual-window
      - Update hint_hz only when r_hint.valid and r_hint.reason == "none" (per D8 / SPEC D8):
          hint_hz = r_hint.freq_hz
        (HINT_MEASURED reason = "hint" must NOT update hint_hz — D8 constraint)
      - Classify each window:
          orig_ok   = r_orig.valid
          hint_ok   = r_hint.valid
          Dropout = not valid (valid=False regardless of reason)
    Return dict: {file, total_fwd, orig_dropout_count, hint_dropout_count,
                  orig_dropout_pct, hint_dropout_pct, improvement_pp}

  For SIN files: use hint_reset_on_dir=True (direction doesn't change in SIN sweeps).

POOLED TOTALS:
  Sum total_fwd, orig_dropout_count, hint_dropout_count across all files (True variant).
  pooled_orig_pct  = orig_dropout_count_total / total_fwd_total * 100
