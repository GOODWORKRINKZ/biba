**Step 2 — rpm_spectral_estimator.c:**

A. Change function definition signature to match (add float hint_hz parameter).

B. Insert the hint-window block BEFORE the final `return result;` statement.
   The hint block must only fire when:
     hint_hz != 0.0f AND fabsf(hint_hz - target_hz) > BIBA_RPM_SPECTRAL_ABS_BAND_HZ / 2.0f

   Inside the hint block (reusing mean, bin_hz, k_max already in scope — no extra scan):
   1. Clamp hint_hz: h_target = clampf_local(hint_hz, BIBA_RPM_SPECTRAL_MIN_TARGET_HZ, BIBA_RPM_SPECTRAL_MAX_TARGET_HZ)
   2. Compute h_half = h_target * BIBA_RPM_SPECTRAL_REL_BAND; if h_half < BIBA_RPM_SPECTRAL_ABS_BAND_HZ then h_half = BIBA_RPM_SPECTRAL_ABS_BAND_HZ
   3. h_lo = clampf_local(h_target - h_half, BIBA_RPM_SPECTRAL_MIN_TARGET_HZ, BIBA_RPM_SPECTRAL_MAX_TARGET_HZ)
   4. h_hi = clampf_local(h_target + h_half, BIBA_RPM_SPECTRAL_MIN_TARGET_HZ, BIBA_RPM_SPECTRAL_MAX_TARGET_HZ)
   5. hk_lo = (uint16_t)ceilf(h_lo / bin_hz); hk_hi = (uint16_t)floorf(h_hi / bin_hz)
   6. Guard: if (hk_lo < 1u) hk_lo = 1u; if (hk_hi > k_max) hk_hi = k_max; if (hk_hi < hk_lo) skip block
   7. Scan bins hk_lo..hk_hi with goertzel_amp_lsb() to find h_best_bin, h_best_amp, h_second_amp
   8. Best-of-two gate: only proceed if h_best_amp >= BIBA_RPM_SPECTRAL_MIN_PEAK_AMP_LSB AND h_best_amp > result.peak_amp_lsb
   9. If gate passes: compute h_noise (same noise floor pattern as plant window but for hk_lo..hk_hi range around h_best_bin), run parabolic interpolation for h_best_bin, then overwrite result fields:
        result.peak_amp_lsb   = h_best_amp
        result.second_amp_lsb = h_second_amp
        result.quality        = h_best_amp / (h_noise + 1.0f)
        result.candidate_hz   = ((float)h_best_bin + h_delta) * bin_hz
        result.freq_hz        = result.candidate_hz
        result.invalid_reason = BIBA_RPM_SPECTRAL_HINT_MEASURED
        result.valid          = true

   Exact C implementation pattern is in RESEARCH.md "rpm_spectral_estimator.c — Implementation extension" —
   use that sketch directly. Do NOT add extra mean computation loops; mean is already declared above.
